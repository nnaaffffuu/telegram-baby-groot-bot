## Installation
```bash
sudo apt-get install python3-pip
sudo pip3 install virtualenv
git clone https://github.com/nnaaffffuu/telegram-groot-bot.git
cd telegram-groot-bot
virtualenv venv
source venv/bin/activate
pip3 install -r requirements.txt
python3 main.py
```
